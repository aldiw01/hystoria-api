# [API] Hysteria BOT
This project is under development, feel free to contact me for question.

## Installation

#### Clone the repo
``` bash
$ git clone https://github.com/aldiw01/hysteria-api.git
```

#### Go into app's directory
``` bash
$ cd hysteria-api
```

#### Import database hysteria_bot.sql into mySQL server

#### Install node and npm
For windows version, you can get it here -> https://nodejs.org/en/download/ 

#### Install app's dependencies
``` bash
$ npm install
```

## Set Environment Variables

#### Primary
```
APP_BOT_TOKEN
APP_DATABASE_DB
APP_DATABASE_HOST
APP_DATABASE_PASSWORD
APP_DATABASE_USER
APP_EMAIL_NAME
APP_EMAIL_PASSWORD
APP_EMAIL_USER
APP_EMAIL_RECEIVER
APP_NAME
```

## Deploy API
``` bash
$ node src/index
```
or
``` bash
$ npm start
```
or
``` bash
$ yarn start
```
